The project rar file contains:

decision_tree_making.py which is the main python file to process the data and get final output.

data_cleaning.py is a helper function for decision_tree_making.py to clean the raw data.

raw-data is a excel file that has 3,001 rows and 10 columns.

final project pdf is the final report with word counts of 1800.

And there is a decision_tree.png showing the final model of the project.
